ShopSphere – Modern E-Commerce Web App

ShopSphere is a modern front-end E-Commerce web application built using HTML, CSS, and JavaScript.
It includes user authentication, product listing, cart management, and order functionality.

This project demonstrates complete JavaScript concepts including:

DOM Manipulation

Event Handling

LocalStorage

Conditional Rendering

Page Protection (Auth Guard)

Dynamic UI Updates
